# 10CT2 Task 1 UX Design - Project Documentation
## Project Proposal
### Design Brief
The book I am using to create my project with is 'Immune' by Kurzgesagt - the popular youtube science channel. The book is an informative biology hardcover that discusses how the immune system changes our bodies using captivating illustrations and story telling. I plan to make a website experience based around the book, targeting Kurzgesagt fans and biology people. The website will also remain informative just like the book but also interactive with the same illustrations.
