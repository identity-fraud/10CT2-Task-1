# 10CT2 Task 1 UX Design - Project Documentation
## Project Proposal
### Design Brief
The book I am using to create my project with is 'Immune' by Kurzgesagt - the popular youtube science channel. I chose this book because I have always loved the Kurzgesagt channel and their series about the immune system and had bought the book after watching their related video, so this wouuld be a good opportunity to create an interactive site about it. The book is an informative biology hardcover that discusses how the immune system changes our bodies using captivating illustrations and story telling. I plan to make a website experience based around the book, targeting Kurzgesagt fans and biology people though the site should still be understandable to someone who doesn't know anything about biology or science, though for the best user experience it is recommended 12+ because of the reading comprehension required. Because this project is educational while being engaging it would appeal to many of the targeted audience The website will also remain informative just like the book but also interactive with the same illustrations ensuring that it remains interesting and very educational.
