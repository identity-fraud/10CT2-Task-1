# 10CT2 Task 1 UX Design - Project Documentation
## Project Proposal
### Design Brief
The book I am using to create my project with is 'Immune' by Kurzgesagt - the popular youtube science channel. I chose this book because I have always loved the Kurzgesagt channel and their video series about the immune system convinced me to buy their corresponding book, so this wouuld be a good opportunity to create an interactive site about it. The book is an informative biology hardcover that discusses how the immune system changes our bodies using captivating illustrations and story telling. I plan to make a website experience based around the book, targeting Kurzgesagt fans and biology people though the site though it should still be understandable or accessible to someone who doesn't know much about biology or science, though for the best user experience it is recommended for ages 12+ because of the reading comprehension required. Because this project is educational while being engaging it would appeal to many of the targeted audience and other people wanting to learn about the immune system in general, the website will also remain informative just like the book but also interactive with the same illustrations ensuring that it remains interesting and very educational. I plan to use HTML and CSS to create my site, hosted through Github Pages, HTML and CSS allows me to fully customise my site without limitations unlike other website designers such as Figma, and Github Pages integrates this with Git allowing me to use Github Desktop for full version control and for my repository which is required for the assessment task.
